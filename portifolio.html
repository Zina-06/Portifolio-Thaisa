Durante o curso de Front End, aprendemos a construir páginas web com HTML, criar layouts responsivos e estilizar elementos usando CSS, adicionar interatividade com JavaScript. Confira alguns projetos:

O HTML(Hypertext Markup Language) é a linguagem de marcação padrão usada para criar a estrutura básica de páginas web. Ele permite organizar e formatar conteúdo como textos, imagens, links e outros elementos, formando a espinha dorsal de qualquer página na internet. Exemplo: PROJETO EURO
O CSS (Cascading Style Sheets) é uma linguagem de estilo usada para definir a aparência e o layout de páginas web criadas com HTML. Ele permite controlar aspectos como cores, fontes, espaçamentos e posicionamento de elementos, tornando o design das páginas mais atraente e responsivo. Exemplo: Grid Layout
O JavaScript é uma linguagem de programação usada para adicionar interatividade e dinamismo às páginas web.É possível criar funcionalidades como menus suspensos, sliders, validação de formulários e outras ações em tempo real.
